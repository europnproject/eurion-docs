# FAQ (Draft)

## Eurion OS è una distro Linux?
Eurion usa una base Linux immutabile per affidabilità e sicurezza.
Ma l’esperienza è progettata come prodotto consumer.

## Avrà compatibilità Windows?
Sì. È un requisito.
Inizialmente useremo un layer guidato (wizard) e profili preconfigurati.

## Telemetria?
Possibile solo nel rispetto GDPR:
- opt-in
- preview payload
- minimizzazione
- cancellazione
